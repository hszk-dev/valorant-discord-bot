プライバシーポリシー

最終更新日: 2025年7月16日

1. 基本方針

Valorant Discord Bot（以下「本サービス」）では、お客様の個人情報保護を重要視し、個人情報保護法およびその他関連法令を遵守して、適切な取り扱いを実施します。

2. 収集する情報

2.1 基本情報

データ種別 具体的内容 収集目的
Discord情報 ユーザーID、ユーザー名、アバター サービス提供、ユーザー識別
サーバー情報 Guild ID、チャンネル情報 機能提供、設定管理
利用ログ コマンド使用履歴、アクセス時間 サービス改善、不正検知
エラーログ エラー内容、発生時刻 問題解決、品質向上

2.2 自動収集情報

  • IPアドレス（セキュリティ目的のみ）
  • 利用時間・頻度
  • パフォーマンスデータ
  • システム利用状況

2.3 収集方法

  • 直接入力: コマンド実行時の情報入力
  • Discord API: Discord連携による自動取得
  • 自動ログ: システム利用時の自動記録

3. 情報の利用目的

3.1 サービス提供

  • Bot機能の提供・実行
  • ユーザー認証・権限管理
  • 統計情報の表示
  • トーナメント管理機能

3.2 サービス維持・改善

  • システムの安定稼働・監視
  • 不正利用・セキュリティ脅威の検知
  • サービス品質向上・新機能開発
  • ユーザーサポート・問い合わせ対応

3.3 法的要求への対応

  • 法令に基づく開示要求への対応
  • 権利侵害・不正行為の調査・対応

4. 情報の共有・提供

原則として第三者への提供は行いません。

4.1 例外的な提供

以下の場合のみ、必要最小限の情報を提供する場合があります：

4.1.1 法的義務

  • 法令による開示義務
  • 裁判所命令・捜査機関要請
  • 生命・身体の安全確保

4.1.2 サービス運営上必要な場合

  • Discord API: 機能提供のみ
  • クラウドサービス: データ保存・処理のみ
  • 統計情報: 個人を特定できない形で集計

5. データの保存・管理

5.1 保存場所・期間

データ種別 保存場所 保存期間
ユーザー情報 暗号化データベース サービス利用停止まで
利用ログ ログファイル 3ヶ月間
エラーログ モニタリングシステム 1ヶ月間
統計データ 集計データベース 匿名化後永続保存

5.2 セキュリティ対策

実装しているセキュリティ対策：

  • 暗号化: 保存データの暗号化処理
  • アクセス制御: 開発者のみアクセス可能
  • 監査ログ: データアクセスの記録・監視
  • 定期バックアップ: データ保護・復旧対策
  • 脆弱性対策: 定期的なセキュリティ更新

5.3 データ削除

  • サービス退会時: 個人データの完全削除
  • サービス終了時: 30日以内に全データ削除
  • お客様要求時: 合理的期間内にデータ削除

6. Cookie・追跡技術

6.1 使用する技術

本サービスは以下の技術を使用します：

  • セッション管理: Discord認証情報の一時保存
  • ログ収集: サービス利用状況の分析
  • エラー追跡: 問題の特定・解決

6.2 用途・制御

  • サービス提供・ユーザビリティ向上
  • システム監視・品質改善
  • セキュリティ脅威の検知

7. 国際間のデータ移転

7.1 移転の発生

以下のサービス利用により、データが海外に移転される場合があります：

  • Discord API: アメリカ（Discord Inc.）
  • クラウドサービス: 適切な保護措置を実施

7.2 保護措置

  • 適切な契約・約款による保護
  • データ保護規制の遵守
  • セキュリティ基準の確認

8. お客様の権利

8.1 データに関する権利

  • アクセス権: 保存されている個人データの確認
  • 訂正権: 誤った情報の修正・更新
  • 削除権: 個人データの削除要求
  • 利用停止権: データ処理の停止要求

8.2 権利の行使方法

以下の方法でお申し出ください：

  • Email: hszk.dev@gmail.com
  • GitHub: Issue作成

8.3 対応期間

  • 確認・訂正: 14日以内
  • 削除: 7日以内
  • 複雑な案件: 30日以内（進捗報告あり）

9. 未成年者の保護

9.1 年齢制限

  • 13歳未満: サービス利用不可
  • 18歳未満: 保護者同意が必要
  • 適切な監督: 保護者による利用監督

9.2 保護措置

  • 年齢確認の実施
  • 不適切コンテンツのフィルタリング
  • 保護者への情報提供

10. プライバシーポリシーの変更

10.1 変更の通知

  • 重要な変更: 30日前の事前通知
  • 軽微な変更: ウェブサイトでの公表
  • 緊急時: 可能な限り迅速な通知

10.2 変更の有効性

通知後の継続利用をもって変更への同意とみなします。同意できない場合は、サービス利用停止を推奨します。
